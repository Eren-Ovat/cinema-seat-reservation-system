# Sinema Koltuk Rezervasyon Sistemi

Bu proje, kullanıcıların sinema filmlerini görüntüleyebileceği, boş koltukları seçerek rezervasyon yapabileceği kapsamlı bir web uygulamasıdır.

## 🚀 Özellikler
- **Kullanıcı Arayüzü:** Angular ile geliştirilmiş modern ve duyarlı (responsive) tasarım.
- **Rezervasyon Yönetimi:** Film seçimi, seans takibi ve interaktif koltuk seçimi.
- **Yönetim Paneli:** Yeni film ekleme, salon kapasitesi belirleme ve rezervasyon kayıtlarını izleme.
- **Backend:** Veri yönetimi için PHP ve MySQL entegrasyonu.
